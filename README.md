# SENTINEL-LITE 🛡️

A lightweight Linux Endpoint Detection and Response (EDR) system built for learning and experimentation.

## Features

* Process monitoring via /proc
* Suspicious behavior detection
* Reverse shell detection
* Modular detection engine

## Roadmap

* [x] Process scanner
* [ ] Real-time monitoring
* [ ] eBPF integration
* [ ] Response engine (kill/isolate)

## Tech Stack

* Python (initial)
* C / eBPF (planned)

## Goals

This project is built to deeply understand:

* Linux internals
* Offensive techniques (for defense)
* Detection engineering

## Disclaimer

This tool is for educational purposes only.
